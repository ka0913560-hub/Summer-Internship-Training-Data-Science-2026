## 🥧 Pie Chart vs Donut Chart

### Pie Chart
- Displays data as slices of a circle.
- Best for showing proportions of a whole.
- Suitable when there are fewer categories.

### Donut Chart
- Similar to a Pie Chart but contains a hole in the center.
- Provides better readability when displaying multiple categories.
- The center area can be used to display additional information.

# 🧮 DAX (Data Analysis Expressions)

Learned the basics of DAX, the formula language used in Power BI for creating calculations and business logic.

### Types of DAX

### Implicit DAX
- Automatically generated by Power BI.
- Used when numeric fields are directly added to visuals.

Example:

```DAX
SUM(Sales)
```

### Explicit DAX
- Written manually by the user.
- Used for custom calculations and advanced analysis.

Example:

```DAX
Total Sales = SUM('Sample - Superstore'[Sales])
```

## ⭐ Star Schema

- One Fact Table connected to multiple Dimension Tables.
- Simple and easy to understand.
- Faster query performance.
- Most commonly used in Power BI.

---

## ❄️ Snowflake Schema

- Dimension tables are further normalized into additional tables.
- Reduces data redundancy.
- Requires more joins.
- Slightly more complex than Star Schema.
