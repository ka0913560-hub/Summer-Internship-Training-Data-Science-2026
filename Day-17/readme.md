## 📌 CNN Introduction

Convolutional Neural Network (CNN) is a specialized type of Deep Learning model mainly used for image processing and computer vision tasks.

Unlike ANN, which works well with structured/tabular data, CNN automatically extracts important features from images using convolution and pooling operations.

### Common Applications

- Image Classification
- Face Recognition
- Object Detection
- Medical Image Analysis
- Handwritten Digit Recognition
- Self-Driving Cars

### Basic CNN Architecture

```
Input Image
      ↓
Convolution Layer
      ↓
ReLU Activation
      ↓
Pooling Layer
      ↓
Flatten Layer
      ↓
Fully Connected Layer
      ↓
Output Layer
```
