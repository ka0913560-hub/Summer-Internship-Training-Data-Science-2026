# 📌 1. Principal Component Analysis (PCA)

## 📚 Overview
Principal Component Analysis (PCA) is an Unsupervised Machine Learning technique used for Dimensionality Reduction. It transforms high-dimensional data into a smaller number of Principal Components while preserving the maximum possible information (variance).

## 🎯 Objectives
- Reduce the number of features
- Preserve maximum variance
- Improve computational efficiency
- Visualize high-dimensional datasets in 2D

## 🛠️ Workflow
- Load Dataset
- Data Standardization
- Apply PCA
- Transform Features
- Analyze Explained Variance
- Visualize Principal Components

## 📂 Dataset Used
- Iris Dataset (Scikit-learn)

## 📚 Libraries Used
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## 🔑 Key Concepts
- Dimensionality Reduction
- Principal Components (PC1, PC2)
- Variance
- Explained Variance Ratio
- Feature Transformation

# 🤖 2. Reinforcement Learning

## 📚 Overview
Reinforcement Learning is a Machine Learning paradigm where an Agent learns by interacting with an Environment through trial and error to maximize cumulative rewards.

## 🎯 Objectives
- Understand Agent-Environment interaction
- Learn reward-based decision making
- Implement Q-Learning
- Implement SARSA
- Compare both reinforcement learning algorithms

## 🛠️ Workflow
- Create Environment
- Initialize Q-Table
- Select Actions
- Receive Rewards
- Update Q-Values
- Train Agent
- Test Learned Policy

## 🎮 Environment Used
- FrozenLake-v1 (Gymnasium)

## 📚 Libraries Used
- NumPy
- Gymnasium

## 🔑 Key Concepts
- Agent
- Environment
- State
- Action
- Reward
- Q-Table
- Exploration vs Exploitation
- Epsilon-Greedy Policy

## 📌 Algorithms Covered
- Q-Learning (Off-Policy)
- SARSA (On-Policy)
